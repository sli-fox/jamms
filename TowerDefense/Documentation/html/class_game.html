void Game::gameLoop ( )

Drives the game.

Returns
Void.

This function controls the speed of the logic in the game and passes it to any function that needs it. since gameLoop() is called every frame, this function starts the clock at the beginning of the frame, stops it at the end and calculates the elapsed time.

The functions from GameState are then called to draw updates to the render window.
